# HomeHelp Business Rules

## Accounts

-   One account per person
-   Multiple roles supported
-   Multiple services supported

## Provider Rules

-   Phone verification required
-   ID required
-   At least one guarantor required
-   Manual admin approval required
-   Only approved providers appear in search

## Contact

-   In-app chat first
-   WhatsApp, Telegram and Phone are optional
-   External contact available after accepted application or booking

## Admin

-   Only Super Admin creates Super Admins
-   Admin actions are audited

## Ethiopia MVP

-   No background checks
-   Manual ID verification
-   Manual guarantor verification
