# HomeHelp Project Context

## Mission

HomeHelp is a trusted household services platform connecting Ethiopian
families with verified household service providers in Addis Ababa, with
future expansion across Ethiopia.

## Technology

-   Flutter
-   Firebase Authentication
-   Cloud Firestore
-   Git
-   GitHub

## Architecture

-   HomeHelp Architecture v1.2 (Frozen)
-   One Firebase UID per person
-   Multiple roles per user
-   Multiple services per provider
-   Service layer between UI and Firebase
-   Trust-first design

## Verification

-   Phone verification
-   ID verification
-   Guarantor (ተያዥ) verification
-   Manual admin approval

## Working Rules

-   Keep existing UI and project structure.
-   Work one step at a time.
-   State modified files before coding.
-   State whether replaced code is needed later.
