# HomeHelp Architecture v1.2 (Frozen)

## Vision

HomeHelp is a trusted household services platform connecting Ethiopian
families with verified household service providers.

## Principles

-   One Firebase UID per person
-   Multiple roles per user
-   Multiple services per provider
-   Service layer between UI and Firebase
-   Trust before growth
-   Soft delete
-   Audit trail
-   Evidence-driven architecture changes

## User Types

-   Family
-   Service Provider
-   Support
-   Admin
-   Super Admin

## Authentication

Current: - Phone - Email/Password

Future: - Google - Apple

## Trust Model

-   Phone verification
-   ID verification
-   Guarantor (ተያዥ)
-   Reviews
-   Completed jobs
-   Profile completion

## Provider Approval

Register → Verify Phone → Complete Profile → Upload ID → Add Guarantor →
Submit → Admin Review → Approved → Searchable
